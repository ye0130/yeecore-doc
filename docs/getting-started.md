## 调试

### 调试表达式

指令：
/yeecore run <玩家> <表达式>

例如：
/yeecore run Yeezhi check(permission("vip.vip1"),"{prefix}§7权限不满足")



### 调试运算结果
指令：
/yeecore eval <玩家> <语句>

例如：
/yeecore eval Yeezhi 四则运算等计算
